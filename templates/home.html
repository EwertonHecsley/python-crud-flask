Bem-vindo ao Meu Projeto Flask

Este é um projeto simples desenvolvido com Flask, um micro-framework web escrito em Python.

O Flask é conhecido por sua simplicidade e facilidade de uso, sendo uma ótima opção para o desenvolvimento de aplicações web.

Neste projeto, você aprenderá a criar funcionalidades CRUD (Create, Read, Update, Delete) para gerenciar produtos.
